What is this site?

This website generates random YouTube videos by creating randomized 5 character strings, and searching YouTube for these strings. It adds these videos to a database where things like the title, and URL are stored.

Why?

I don't know man, it's just a slightly amusing way to pass time. This site is generally meant to be for personal use, and isn't really intended to be used by anyone other than myself and a few friends.

If it's intended to be private then why can I use it?

Because it hasn't become a problem yet, if a lot of people start using the site, and it becomes a problem, I will likely lock the website.

How can I create my own YouTube video generator?

The basis of how this website generates videos can be found on a StackOverflow post. It's just a couple of simple PHP functions and isn't hard to follow.
